jquery.imgExplosion Demos

Using Defaults

The library comes with a star graphic, assumed stored in /img/star.png, which it uses as the default:

Simplest Implementation

	imgExplosion();

Attaching to an image on the page

Throw at an angle

	$('#Image1').click(function(){
		$('#Ninja').imgExplosion({
			angle:true
		});
	});

Alternate throwing top/bottom

	$('#Image2').click(function(){
		$('#Circle').imgExplosion({
			alternate:true
		});
	});

Alternate at angle

	$('#Image3').click(function(){
		$('#Shocked').imgExplosion({
			angle:true,
			alternate:true
		});
	});
